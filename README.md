## Descripción del proyecto

Este proyecto consiste en una tienda online desarrollada con HTML, CSS, TypeScript y Vite.

La aplicación permite visualizar un catálogo de productos, realizar búsquedas dinámicas, filtrar productos por categorías y gestionar un carrito de compras utilizando localStorage para mantener la información guardada en el navegador.

Entre las funcionalidades implementadas se encuentran:

- Renderizado dinámico de productos
- Búsqueda en tiempo real
- Filtros por categorías
- Agregar productos al carrito
- Aumentar y disminuir cantidades
- Eliminar productos
- Vaciar carrito
- Persistencia de datos con localStorage
- Diseño responsive utilizando CSS Grid y Flexbox

## Instrucciones para ejecutar el proyecto

1. Clonar el repositorio:

git clone https://github.com/matfarfan/programacion-III---TPI.git

2. Instalar dependencias:
pnpm install

3. Ejecutar el proyecto:
pnpm dev

4. Abrir el navegador en la URL indicada por Vite.
